# CapsNet(pytorch)

CapsNet based on Geoffrey Hinton's original paper 
[Dynamic Routing Between Capsules](https://arxiv.org/abs/1710.09829)

## Requirements

 - python3
 - pytorch
 - sklearn

```bash
pip3 install torch torchvision

pip3 install ipywidgets jieba gensim tqdm sklearn
```

## Run

```bash
python3 train.py
```
